影響體驗的四大因素

  • 便利性
  • 反應速度
  • 外觀體驗
  • 品牌形象

便利性

  1. weight : 隨時一手帶著到處跑的輕盈
  2. size : 隨時塞入手提包，不會太大、太小
  3. battery : 隨時不中斷的電力支援(10hr up)
  4. I/O : 隨時想插什麼線，都有孔可以支援

反應速度

  • Ram : 暫存記憶體反應速度及大小(2100Hz, 8G up)
  • Storge : 硬碟容量大小(128G ssd + 1T hdd up)
  • CPU : 中央處理器速度(i5系列 up
  • (選) GPU : 有玩大型遊戲、渲染高畫質影片需求(GTX-1050 up)
